■遊び方
------------------------------------------------------------------------
●ゲームルール
ジャンル：ルーレット式コマンドカードバトル
ルーレットを止めて、モンスターのコマンドを決定。
敵を倒して勝利を目指しましょう。

〇モンスターのステータス・技一覧
「デッキ編集」より見ることができます。
技名にマウスを合わせると、技の詳細を見ることができます。
------------------------------------------------------------------------

●操作方法
------------------------------------------------------------------------
基本操作：マウス操作のみ

〇左クリック
次に進む
ボタンの決定
ルーレットの停止
攻撃対象の決定

〇ホイール
ステージ、戦闘ユニットの選択
攻撃対象の選択

〇Escapeキー
ゲーム終了：ウィンドウを閉じる

〇Spaceキー
バトルシーン中に押すと、モード選択画面に行きます。
バトルが泥試合になったら使ってください。
------------------------------------------------------------------------
