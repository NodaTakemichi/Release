■概要
------------------------------------------------------------------------
ジャンル	：3D脱出ホラーゲーム
タイトル	：「縫い包み」
プレイ人数	：1人
プラットフォーム：PC
------------------------------------------------------------------------


■遊び方
------------------------------------------------------------------------
『不気味な館から脱出しよう』
その館には、不気味な徘徊する化け物がおり、
不思議な力で館の外に出ることが出来ません。
護符やぬいぐるみのアイテムを使い、時間を
稼いだり、道を切り開くことで恐怖の館から脱出しましょう。
------------------------------------------------------------------------


■操作方法
------------------------------------------------------------------------
コントローラー：キーボード、マウス

・視点操作	　：マウス
・調べる	　：左クリック
・アイテム使用	　：右クリック
・移動		　：WASDキー
・走る		　：左Shiftキー

・ポーズメニュー　：Escapeキー
・ボタン決定	　：Spaceボタン
・ボタン選択	　：WSキー
------------------------------------------------------------------------
